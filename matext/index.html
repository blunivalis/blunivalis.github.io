MaText

MaText is a simple Android app for converting, cleaning up, reformatting, and stylizing plain text.

Features

  • Type or paste text into a simple text workbench
  • Choose local text tools for case changes, cleanup, lists, formats, and fun effects
  • Copy transformed output or use it as the next input
  • Ad-supported free version with a strong set of everyday text tools
