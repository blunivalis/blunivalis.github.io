MaFocus

MaFocus is a simple Android app for structured focus sessions, short breaks, and longer recovery breaks.

Features

  • Start focus, short break, and long break timers
  • Create, edit, and manage focus presets
  • Track basic focus statistics in a simple interface
  • Import and export local app data for offline backup
